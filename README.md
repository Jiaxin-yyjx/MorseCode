# MorseCodeMastery

Open the same .venv in Flask backend

MacOS:
```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
```
Windows:
```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```
